# MediaPlayer 交互规范 V1.5

## 主体布局

- 根布局固定为 `Header + Sidebar + Main`。
- Header、Sidebar、Main 的比例均可由用户调整。
- Sidebar 与 Main 之间提供可拖动分割条，用于实时调整布局比例。
- Sidebar 宽度低于 `3%` 时自动折叠，仅显示单个“展开目录”按钮。
- 折叠态“展开目录”按钮位于 Sidebar 容器垂直居中，箭头朝右，颜色为灰色，透明度从左到右递减；悬停显示文字提示。
- Main 区包含：
  - 分页内容区
  - 可折叠元数据面板
  - 当前 focus 项信息 Footer

## 当前阶段实现说明（虚拟 UI）

- 当前阶段使用虚拟数据驱动交互，不加载真实图片/视频文件。
- 图片以色块与分辨率文本占位，视频以数值时钟和状态文本占位。
- 无法直接可视化的动作允许以控制台输出作为反馈。
- 验收目标是“布局、控件位置、快捷键作用域、状态联动”与本文档一致。

## Header 控件

- Logo 作为统一导入入口。
- Logo 菜单提供“导入文件”“导入文件夹”两类弹窗入口。
- 混合输入通过“全窗口拖拽”与“粘贴路径”实现。
- 检测到文件/目录拖拽进入窗口时，显示全窗口导入叠加层（当前为占位层）。
- 模式切换：图片模式 / 视频模式。
- 检索入口：Header 仅保留“检索”按钮（视频模式禁用）。
- 点击“检索”按钮展开检索容器（复用原向量检索容器）。
- 检索容器顶部提供“向量检索 / 特征检索”切换按钮（形态与 Header 模式切换一致）。
- 检索容器支持折叠；折叠后不退出检索模式，以上方居中“下箭头”按钮恢复展开。
- 向量检索页包含“向量检索”按钮与“相似度阈值”滑条。
- 特征检索页改为多字段输入：`名称`、`作品名`、`社团`、`作者`、`tags`、`图包评分`。
- `名称`、`作品名` 支持模糊匹配。
- `社团`、`作者` 输入框基于库内值提供自动补完候选。
- `tags` 通过标签选择面板点选（支持多选）。
- `图包评分` 通过 `0~5` 六个星级选项筛选；选择第 N 项时，前置项同步高亮。
- 多字段组合采用 AND 逻辑过滤结果；单字段时仅按该字段输出结果。
- 检索容器与下方工作区之间提供可拖动分割条，用于调节检索容器高度。
- 向量检索/特征检索切换时，检索容器高度会自动贴合当前控件所需空间。
- 向量检索输入来源于当前缩略图中已选中的图片。
- 执行检索后，结果第 1 张固定为输入图片，后续按相似度降序展示并显示相似度数值。
- 相似度阈值修改后不自动刷新结果，需再次点击“向量检索”才更新结果集。
- 当前图包评分控件。
- 缩略图离散缩放控件（固定 9 级，Header 形态：`缩放级别 - 当前级别数字 +`）。
- 自动播放开关与间隔预设。
- 设置面板入口。

## Sidebar 行为

### 图片模式

- 展示分层根目录、子目录与图包节点。
- 不展示图片文件叶子。
- 任意节点可设置为 `Current Root`，用于范围化浏览。
- 目录结构栏在“未设置根”时显示统一“设为根”按钮，仅在当前选中节点为目录时可用。
- “已设置根”时隐藏“设为根”，右侧仅显示“恢复”按钮用于回到数据库根。
- 若已设置根目录，标题区直接显示根目录名称；标题可点击并直接折叠 Sidebar。
- 目录节点右侧显示“当前目录直接包含的图片数”，不累计其子目录与压缩包内目录。
- 压缩包中的目录按同样规则展示直接图片数。
- 当目录仅包含图片时，Main 直接显示该目录对应图片缩略图。
- 主界面分割条（Sidebar/Workspace、检索容器/主区、Main/Metadata）受“布局锁定”开关控制：锁定时不可拖动。
- 检索结果视图下，Sidebar 内容保持“路径/节点树”形态，不重复列出同一节点；仅最末级命中叶节点显示检索结果数量，上级目录不累计显示。
- 向量检索结果视图下，Sidebar 左侧点击与方向键选择不驱动右侧 focus；右侧缩略图 focus 变化时，Sidebar 仅同步高亮对应节点。
- 特征检索结果视图下，Sidebar 行为与常规模式一致：可选中节点并驱动右侧缩略图内容。
- 检索结果视图下，若 Sidebar 可视区外的节点被同步高亮，Sidebar 需自动滚动到可见位置。
- 检索结果视图下，Sidebar 头部将“设为根”替换为“转到”；点击“转到”后退出检索模式，恢复常规 Sidebar 排序，并定位到当前 focus 所在节点。

### 视频模式

- 展示分层根目录、子目录与视频文件叶子。
- 每个视频叶子带 `toggle`，用于加入/移出全屏播放列表。
- 播放列表为全库共用。

## Main 行为

### 图片模式

- 使用分页缩略图网格，不使用无限滚动。
- 支持“缩略图模式”与“纯文件名模式”切换。
- 缩略图模式默认卡片比例为 `3:4`，分页内按行列顺序排布；整块网格居中显示（不改变每行从左到右顺序）。
- 缩略图内图片按长边适配并居中：`landscape / square` 垂直居中，较窄 `portrait` 水平居中。
- 缩放算法采用离散级别：仅 9 级，初始为第 5 级；初始级别以“缩略图宽度设置值的 75%”作为锚点自适应选取。
- 纯文件名模式改为文件列表视图，不分页，改为滚动浏览。
- 支持点击与键盘移动 focus。
- 元数据面板随 focus 实时更新。
- 元数据面板在展开态由标题“元数据面板”本身作为折叠入口；折叠态使用侧边箭头按钮恢复展开。

### 视频模式

- 提供视频预览区域与播放控制。
- 包含 `Save as cover` 按钮，用于手动保存封面。
- 当前阶段 `Save as cover` 保存随机颜色 Mock 封面并输出 `console`。
- 切换 Sidebar 视频项后进入新视频封面暂停态，不继承旧进度。
- 播放控制区包含进度条（可拖动）、静音开关、音量调节与时间显示。
- 播放控制区包含倍速滑条，范围 `0.1x ~ 4.0x`，默认 `1.0x`。
- 元数据面板可切换：
  - 视频信息视图
  - 播放列表管理视图

### Main Footer

- 始终展示 focus 项详细信息。
- 展示项包括路径、文件大小、分辨率。
- `zip` 图片路径格式：`压缩包绝对路径 + 序号`。

## 播放列表交互

- 支持拖拽调整顺序。
- 支持直接删除条目。
- 删除条目后，同步 Sidebar 对应 `toggle` 关闭。
- 若删除当前播放项，自动切换到下一可用项。

## 全屏模式

- Enter 或双击 focus 项进入全屏。
- `F` (`KeyF`) 切换全屏/窗口模式（图片模式与视频模式都可用）。
- 从图片模式进入时默认 `仅图片`；从视频模式进入时默认 `仅视频`。
- 鼠标进入底部 20% 区域时显示“浮动叠加”Footer，不占用内容区高度。
- Footer 保留“双显示”切换与“调换左右”按钮：
  - 手动点击“双显示”后进入双显示。
  - 双显示态可切回单显示；回到单显示时始终恢复“进入全屏时的单显示模式”（图片进入则回图片，视频进入则回视频）。
- 双显示态移除固定分屏比例滑条，改为拖动中间分隔线实时调整比例。
- 双显示态下，focus 通过“点击图片/视频区域”或 `Tab` 切换；Footer 导航与对齐操作作用于当前 focus 区域。
- 双显示态下，视频区域支持鼠标左键拖动；按视频比例自适应后，基于剩余留白空间可进行水平/垂直位置微调。
- 全屏图片/视频按实际宽高比自适应缩放（Mock 占位同样遵循实际比例）。
- 双显示态强制使用容器内自适应缩放，不继承单显示缩放值。
- 单显示态支持缩放与平移：
  - Footer 缩放按钮。
  - `Ctrl + 鼠标滚轮` 缩放。
  - 鼠标左键拖动平移。
  - `Reset` 恢复到自适应缩放。
- 全屏视频区域内，鼠标移入时显示悬浮播放控件，移出后隐藏。
  - 控件仅两种锚点：贴视频底部或贴视频顶部（根据视频在容器中的垂直位置自动切换）。
  - 贴底部时顺序为“进度行 -> 播放控制行”（进度行贴近视频底边）。
  - 贴顶部时顺序为“播放控制行 -> 进度行”（进度行贴近视频顶边）。
- Footer 增加“上个包/下个包”按钮（图片链路）。

## 键盘控制

### 全局与图片浏览

- `Left/Right`：上一张 / 下一张。
- 非全屏 `Up/Down`：按网格上移 / 下移一个可视行步长；若越界则钳制到首项 / 末项。
- 全屏 `Up/Down`：跳到当前包首张 / 末张（不走网格步长）。
- `Ctrl+Left/Right`：上一个包 / 下一个包。
- 全屏图片态下，`Ctrl+Left/Right` 在 Footer 控件获得焦点时仍强制生效（不被输入控件吞掉）。
- 全屏下 `Alt+Up/Down/Left/Right`：对当前 focus 区域执行上/下/左/右对齐。
- `A`：开始 / 停止自动播放。
- `1/2/3/4/5`：自动播放间隔 `1/2/3/5/8` 秒。
- `Alt+0..5` 或小键盘 `0..5`：图包评分（`0 => null`）。
- 非全屏下 `Tab`：在 Sidebar 与 Main 之间切换键盘焦点。
- 全屏且双显示下 `Tab`：在图片区域与视频区域之间切换 focus。
- 全屏图片翻页到边界时，继续按上一张/下一张会跨包进入上一个/下一个包；自动播放遵循同规则。
- 非全屏且焦点在 Sidebar 时，`方向键/Enter/PageUp/PageDown/Home/End` 用于 Sidebar 导航，不作用于缩略图翻页。
- `ArrowUp/ArrowDown` 移动到不可见项时，Sidebar 自动滚动保证目标项可见。
- `PageUp/PageDown` 以“当前可视项数量 - 1”作为步长翻页；超出边界时直接钳制到首项/末项。
- 鼠标点击 Sidebar 项后，焦点保持在 Sidebar；仅点击缩略图区或使用 `Tab` 才切回 Main 焦点。

### 视频控制

- 仅在“全屏视频”或“双显示且视频焦点激活”时生效。
- `Space`：播放 / 暂停。
- `PageUp/PageDown`：播放列表上一个 / 下一个视频。
- `Ctrl+PageUp/PageDown`：降低 / 提高播放倍速。
- `Home/End`：降低 / 提高音量。

## 快捷键自定义

- 所有快捷键均可在设置面板重映射。
- 保存时按作用域执行冲突检测。
- 支持一键恢复默认快捷键方案。

## 设置面板形态

- 以约 80% 的居中面板形式打开。
- 背景内容轻度模糊但保持可见。
- 面板内部采用 `side | main` 分栏：左侧为设置分类，右侧为对应主内容区。
- 默认宽度配比为 `side 20% | main 80%`，横向占满设置面板。
- `main` 区沿用圆角面板外观并占满分配宽度；内部内容容器仅负责居中排版，不增加额外边框层。
- 左侧分类包括：布局参数、模型参数、快捷键设置、theme 设置、3D 设置。
- `theme 设置` 与 `3D 设置` 当前为占位页，预留后续主题与 3D 空间浏览向量库映射配置。
- 布局参数页新增“布局锁定”开关：锁定后禁用主界面分割条拖动（不影响全屏模式拖动行为）。
- 布局参数页新增“设置面板字体大小”滑条。
- 各设置页主标题居中，输入框与滑条优先拉伸以完整显示参数内容。
- Sidebar 相关可调参数包括：最小宽度、主文字字号、数字字号、目录结构间隔。
- 目录结构间隔拆分为：多级目录水平缩进间隔与上下项间距。
