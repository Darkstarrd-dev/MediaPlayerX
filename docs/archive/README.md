# 文档归档目录

本目录用于存放非当前迭代使用、但仍需保留追溯价值的历史文档。

## 子目录说明

- `project-evaluations/`：历史项目评估报告、评估模板与对应产物目录。
- `implementation-plans/`：已完成或已替代的历史实施计划。
- `offline-auto-subtitle/`：离线自动字幕历史 Phase 执行报告。
- `stability-notes/`：历史稳定性问题记录、排查结论与修复复盘。
- `checklists/`：历史阶段性检查清单。

## 使用约定

- 根目录 `docs/` 仅保留 SSOT 与当前迭代文档。
- 新迭代若产生替代文档，旧版本移入对应归档子目录。
