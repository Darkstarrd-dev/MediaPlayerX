# MediaPlayerX

本仓库用于开发本地优先的综合媒体浏览器（图片 + 视频）。

当前阶段：**虚拟 UI 交互阶段**。

- 仅验证 UI 布局、交互、快捷键与状态联动。
- 不接真实数据库、真实媒体解码与真实向量推理。

## 运行项目

```bash
npm install
npm run dev
```

## 运行测试

```bash
npm run test
```
